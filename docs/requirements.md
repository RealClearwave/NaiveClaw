# Requirements

## Dependencies

- Python 3.9+
- `openai`: OpenAI API client (used to interact with LM Studio's OpenAI compatible API)
- `mss`: Fast screenshots
- `Pillow`: Image processing and base64 conversion
- `pyautogui`: Keyboard and mouse control
- `python-dotenv`: Environment variable management

## Installation
```bash
pip install openai mss pillow pyautogui python-dotenv
```
